Detailed Description

Cumulocity HTTP binary API implementation.

Cumulocity binary APIs are used for uploading and downloading big files to and from your Cumulocity file repository.

int SrNetBinHttp::getf ( const string &  id,
const string &  dest 
)

Cumulocity HTTP binary get.

Counterpart of the get API, except this function stores the response directly to a file. Suitable for large binary files.

Note
when using this function, the response can not be accessed by the resp function, rather stored in a file specified by argument path.
Parameters
idCumulocity binary resource unique identifier.
destlocal file path to store the response.
Returns
size of file on success, -1 on failure.
