Detailed Description

SmartREST response parser.

As the SmartREST protocol supports message aggregation, multiple SmartREST messages can be aggregated into one response. This class is designed for parsing one single response, which in turn contains multiple SrRecord.

Get the next SmartREST record.

This function is state-ful, no re-entrant. It parses the entire buffer with a SrLexer for one SrRecord and stops. This function implements an iterator that is commonly found in high-level languages.
